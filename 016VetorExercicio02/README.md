## Problema Exemplo 02

Fazer um programa para ler um número inteiro N e os dados (nome e preço) de N produtos.
Armazene as N produtos em um vetor. Em seguida, mostrar o preço média dos produtos.

### Exemplo
|   Entrada     |  Saída                 |
| ------------- | ---------------------- |
|      3        | AVERAGE PRICE = 700.00 |
|      TV       |                        |
|    900.00     |                        |
|     Fryer     |                        |
|     400.00    |                        |
|     Stove     |                        |
|     800.00    |                        |
