## Problema Exemplo 01

Fazer um programa para ler um número inteiro N e a altura de N pessoa.
Armazene as N alturas em um vetor. Em seguida, mostrar a altura média dessas pessoas.

### Exemplo
|   Entrada     |  Saída                 |
| ------------- | ---------------------- |
|      3        |  AVERAGE HEIGHT = 1.69 |
|      1.72     |                        |
|      1.56     |                        |
|      1.80     |                        |
